# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-25

### Added
- Complete API Telemetry pipeline with Pandas anomaly detection.
- Webhook ingestion engine for GitHub commit and pull request events.
- AI Orchestrator powered by Google Gemini for incident correlation.
- Celery background workers with Redis message broker and Dead Letter Queue pattern.
- High-performance FastAPI backend with SQLAlchemy 2.0 (Async).
- Modern Next.js frontend with Tailwind CSS glassmorphism UI.
- Secure JWT authentication and Workspace isolation.
- Full deployment pipeline documentation for free-tier infrastructure.
- `seeder.py` script for instant "Demo Experience".
